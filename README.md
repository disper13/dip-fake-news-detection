**Мультимодальная модель распознавания фальшивых новостей на основе DistilBERT и ResNet18**

Данный репозиторий содержит код и материалы выпускной квалификационной работы на тему: **«Разработка модели распознавания фальшивых новостей с учётом их мультимодальности на основе машинного обучения»**.


## 📊 О проекте
Система предназначена для автоматического выявления и классификации фейковых новостей путём одновременного анализа **текста** (заголовка) и **изображения**. Проект решает проблему недостаточной эффективности чисто текстовых моделей за счёт учёта мультимодального характера контента в социальных сетях.

### Ключевые особенности
- **Мультимодальный анализ**: одновременная обработка текста и изображений
- **Late Fusion**: механизм слияния модальностей на уровне логитов (поэлементный максимум)
- **Предобученные модели**: использование DistilBERT и ResNet18 с весами ImageNet
- **Балансировка классов**: взвешенная функция потерь для работы с несбалансированными данными
- **Early Stopping**: автоматическая остановка обучения для предотвращения переобучения

##  Архитектура
### Компоненты модели
| Компонент | Архитектура | Описание |
|-----------|-------------|----------|
| **Текстовый энкодер** | DistilBERT Base | 6 слоёв трансформера, 768 скрытых единиц |
| **Визуальный энкодер** | ResNet18 | 18 слоёв, предобучен на ImageNet |
| **Метод слияния** | Late Fusion | Объединение логитов через `torch.max()` |
| **Классификатор** | Fully Connected | Проекция на 6 классов |

### Классы классификации
Модель выполняет 6-классовую классификацию:
1. **TRUE** — достоверная информация
2. **SATIRE** — сатира/пародия
3. **FALSE_CONN** — ложная связь (заголовок не соответствует содержанию)
4. **IMPOSTER** — выдача себя за другое издание
5. **MANIPULATED** — манипуляция с изображением
6. **MISLEADING** — вводящая в заблуждение информация

## 📁 Датасет
Для обучения используется датасет **[Fakeddit](https://github.com/entitize/fakeddit)** — крупнейший публичный датасет для мультимодальной детекции фейковых новостей.

### Характеристики
- **Объём**: свыше 900 000 публикаций из Reddit
- **Формат**: TSV (метаданные) + JPG (изображения)
- **Период**: 2008–2019 гг.
- **Язык**: английский

### Выборка для проекта
В репозитории представлен сэмпл **2% от оригинального датасета** (~10 000 записей) для демонстрации и воспроизводимости результатов. Полный датасет доступен по ссылке выше.

**Примечание**: Изображения не включены в репозиторий из-за ограничений GitHub по размеру. Для обучения модели необходимо:
1. Скачать оригинальный датасет Fakeddit
2. Разместить изображения в папке `/content/drive/MyDrive/VKR/images/` (для Google Colab)
3. Обновить пути в ноутбуке `model.ipynb`

## 🚀 Результаты
Экспериментальное исследование показало следующие метрики на тестовой выборке:

| Метрика | Значение |
|---------|----------|
| **Accuracy** | ~72% |
| **Weighted F1-Score** | ~0.70 |
| **ROC-AUC (macro)** | ~0.91 |

### Метрики по классам
| Класс | Precision | Recall | F1-Score |
|-------|-----------|--------|----------|
| TRUE | 0.77 | 0.83 | 0.80 |
| SATIRE | 0.51 | 0.35 | 0.41 |
| FALSE_CONN | 0.58 | 0.59 | 0.59 |
| IMPOSTER | 0.64 | 0.43 | 0.51 |
| MANIPULATED | 0.81 | 0.79 | 0.80 |
| MISLEADING | 0.68 | 0.68 | 0.68 |

*Наилучшие результаты достигнуты для классов TRUE и MANIPULATED, что подтверждает эффективность мультимодального подхода для выявления манипуляций с изображениями.*
